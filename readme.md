Kata CamelCasing
===
Este programa fue creado como solución para este [ejercicio](https://www.codewars.com/kata/517abf86da9663f1d2000003) de CodeWars.

## Como funciona :mag: 
Ejecutamos el método toCamelCase() de la clase Solution, introduciendo como parámetro una cadena cuyas palabras estén separadas por guiones o barra baja, como por ejemplo: "Hola-Mundo".

El método cambiará todos los símbolos a "_" para luego hacer un split() diviendo las palabras donde hay una barra baja. Después creamos un String que obtendrá las palabras resultantes del split(), habiendo suprimido los símbolos.
